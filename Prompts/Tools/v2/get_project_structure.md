# Tool: get_project_structure

WHAT: Returns the overall structure of the project, showing the directory tree with file and folder names.

WHEN: Use when you need a high-level overview of the project layout, to understand how the codebase is organized, or to find which directories contain relevant source files.

HOW:
- No parameters required. Returns the project tree from the project root.
- Overloading: Not applicable — this tool has no parameters.

OUTPUT: Returns a tree-like structure showing all directories and files, with vendor directories collapsed.
