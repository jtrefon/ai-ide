Infrastructure

Shared services powering both pipelines

CodebaseIndex

SQLite-backed index with FTS5 full-text search, symbol extraction, file watching, and chunk management. The vector index uses HNSW over FAISS for embedding-based semantic search - 10–50× faster than the original brute-force approach with ~95–99% recall.

Services/Index/DatabaseManager.swift

InlineCompletionEngine

Local FIM completion subsystem with ghost-text rendering (Tab accept, Esc dismiss), debounced triggers, language-scoped context assembly, completion caching, duplicate filtering, and ranking. The tightest latency loop in the app - target: <100ms end-to-end.

Services/LocalPipeline/

AITool System

20+ tools implementing the AITool protocol, executed by ToolLoopHandler in the cloud pipeline. Includes file I/O, terminal commands, search, project management, and diagnostic tools. Stall detection prevents runaway loops. Results are QA-reviewed before final application.

Services/Tools/ · Services/CloudPipeline/

ConversationOrchestrator

Multi-role orchestration graph: Planner → Worker → QA → Final. The planner synthesizes a plan, the worker executes tool loops, QA reviews the output, and the finalizer applies the change. Supports streaming responses, conversation folding, and history persistence.

RAG Engine

Cloud-only retrieval pipeline with intent classification, evidence fusion ranking, code segment retrieval (HNSW-backed), symbol search, and project overview. Designed to inject relevant context without overloading the model.

EventBus

Central typed event system using Combine PassthroughSubject. Every pipeline, service, and UI component communicates through the bus. Events are dispatched by type name. Subscribers receive on DispatchQueue.main. Loose coupling by design.

Core/EventBus.swift

Developer experience

Built for the Mac

Syntax highlighting

Powered by tree-sitter AST parsing with the Neon runtime - not regex-based tokenization. Supports 20+ languages with accurate, incremental, and fast highlighting. Customizable color schemes per-language via the settings UI.

Packages/SyntaxHighlighting/

Terminal integration

Embedded terminal with the system shell (/bin/zsh or /bin/bash), build error parsing, and clickable links to editor locations. Build error diagnostics are parsed for file/line references - click to navigate directly to the issue.
