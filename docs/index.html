Architecture

Two pipelines. One editor. Zero compromise.

Local and cloud are completely separate AI pipelines that share the same native editor. Each is tuned for what it does best — so you get cloud-scale power without ever paying for it in latency.

Local pipeline <100ms

Always on. Always private. No network required.

  • Real FIM completion — intelligent ghost text in <100ms
  • Inline AI popover — cursor-anchored Q&A, on your machine
  • Semantic search — HNSW ANN, 10–50× faster, powered by the Neural Engine
  • Direct local model chat & agentic work — fully offline
  • Model: on-device LLM via MLX, accelerated by Apple Silicon

Cloud pipeline Agentic

On demand. Full reasoning. Opt-in privacy.

  • Orchestration graph: Planner → Worker → QA → Final
  • Tool loop with stall detection + recovery
  • RAG context injection with evidence fusion ranking
  • 20+ tools — file I/O, terminal, search, project ops, built-in browser
  • Multi-provider: OpenRouter, DeepSeek, Alibaba, Kilo, local — Anthropic & OpenAI next

Why native wins

Native means something here

Compass is a real macOS binary — compiled in Swift and accelerated by the Metal and Neural Engine frameworks. No Chromium. No Electron event loop burning your CPU. No gigabytes of JavaScript parked in RAM. Just an editor that feels like part of the OS.

~200 MB, AI included

The entire app — with every AI capability switched on — fits in roughly 200 MB. Smaller than most editors manage with AI turned off.

No Electron tax

Tired of an IDE eating 80% of your RAM and slowing to a crawl? No JS loops, no listeners, no unaccelerated UI rendering. Just native performance.

Liquid Glass & deep OS access

Full access to low-level Apple OS features, including the latest Liquid Glass framework — something a web wrapper can never touch.

Under the hood

Engineered for the silicon it runs on

Apple Neural Engine

Embeddings and retrieval run on the ANE — indexing and search stay silent and painless, never stealing cycles from your build.

On-device LLM

Apple MLX running a quantized 1.5B model on the Neural Engine. <100ms end-to-end. Swap models anytime without a website update.

HNSW - FAISS

Hierarchical Navigable Small World index over FAISS vectors. 10–50× faster than brute-force with ~95–99% recall.

SQLite FTS5

Full-text search over indexed symbols, file content, and chunks. Combined with vector retrieval for grounded answers.

Tree-sitter - Neon

Tree-sitter AST parsing with the Neon runtime for fast, accurate syntax highlighting across 20+ languages.

SwiftUI AppKit

Native NSTextView editor with SwiftUI overlays and Liquid Glass. No Chromium. No web view. Just macOS.
