For developers

The editor that finally feels like macOS.

Compass is a Swift-native IDE with real AI built in — completion, retrieval, and agentic coding that run on your Apple Silicon. No Electron. No JavaScript bloat. No code shipped to the cloud.

Real AI, on your machine

Completion that's actually intelligent.

Ghost-text completion in <100ms from a 1.5B on-device model — genuine intelligence, not a fake heuristic. Retrieval runs on the Apple Neural Engine, so indexing and search stay silent and painless while you keep typing.

Offline? Still works. Plane, tunnel, coffee shop — your AI doesn't quit because it never needed the network.

Native to the core

Built for the silicon it runs on.

~200 MB. AI included.

The whole app — every AI capability on — fits in roughly 200 MB. Smaller than most editors with AI turned off.

No Electron tax

A real macOS binary in Swift. No Chromium, no event loop burning your CPU, no gigabytes of JavaScript in RAM.

Liquid Glass & deep OS access

Full low-level Apple OS features, including the latest Liquid Glass framework — something a web wrapper can never touch.

Built-in browser

Read docs and references without leaving the editor. One app, zero context-switching to a browser.

Built-in local inference

MLX-backed models run chat, quick actions, and agentic work on your Mac. Offline. Private. No API key required.

Bring your own model

Route heavy work through OpenRouter, DeepSeek, Alibaba, or Kilo today — Anthropic & OpenAI support lands next iteration.

Build with Compass.

Open source and free. Requires Apple Silicon and macOS 26+.
