Frequently asked questions

What does the closed beta include?

Early access to the commercial hosted tier - bundled inference (no separate API keys needed), priority support, early feature access (remote sessions, multi-file orchestration, project memory), and a say in the roadmap.

Is Compass still open source?

Yes. The core IDE remains open source under the MIT license. The beta enrolls you in the commercial hosted tier - a paid service that bundles inference, removes the need to manage your own API keys, and adds enterprise features.

How much will it cost?

Pricing TBD. We're exploring a usage-based model - not per-seat. Local AI stays free. You only pay for cloud inference and managed features. Beta participants will get a discounted founder rate.

Do I need Apple Silicon?

Yes. Compass requires an Apple Silicon Mac running macOS 26+. The local pipeline uses MLX, the Metal-backed framework designed for Apple's unified memory architecture.

Can I just use the open source build?

Absolutely. Download the latest build from GitHub Releases. You'll need your own API keys for cloud models, but local completion and Q&A work completely offline and free.
